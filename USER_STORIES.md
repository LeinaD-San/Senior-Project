# trAgent — User Stories

## Discovery

- As a user, I want to search a destination (city/neighborhood) so I can start planning a trip.
- As a user, I want to use my current location (optional) so I can discover nearby places.
- As a user, I want the map to update to my current location so I trust the nearby recommendations.
- As a user, I want clear errors when location is denied or unavailable so I know what to do.

## Preferences and results

- As a user, I want to pick interests (coffee, food, parks, etc.) so results match what I like.
- As a user, I want to see a list of results per interest so I can compare options.
- As a user, I want to select specific places so I control my itinerary stops.
- As a user, I want to view place details (photos, address, rating, website) so I can decide if it’s worth visiting.

## Itinerary building

- As a user, I want to set the number of days for my trip so the itinerary is split by day.
- As a user, I want an “auto-generate itinerary” option so I can get a randomized plan quickly.
- As a user, I want to view my itinerary by day so I can understand the schedule.
- As a user, I want to reorder stops within a day so the plan matches my preferences.
- As a user, I want to remove a stop so I can refine my itinerary.
- As a user, I want to add notes to each stop so I remember what to do there.

## Saving and managing trips

- As a user, I want to save a trip so I can come back to it later.
- As a user, I want to reopen a saved trip so I can view/edit it.
- As a user, I want to delete a trip so I can clean up old plans.

## Reliability and security

- As a user, I want the app to handle “no results” gracefully so I’m not stuck.
- As a user, I want my API keys to stay private so my account isn’t abused.

## Accounts

- As a user, I want to create an account with email and password so I can save my itineraries.
- As a user, I want to log in after creating my account so I can access my saved itineraries from any device.
- As a user, I want to stay signed in between page refreshes so I don’t have to log in repeatedly.
- As a user, I want to plan a trip without an account so I can try the app quickly.
- As a user, I want to be prompted to sign up when I try to save an itinerary so I understand why an account is needed.
- As a user, I want to save an itinerary to my account so I can view it later.
- As a user, I want to view a list of my saved itineraries so I can reopen one quickly.
- As a user, I want to open a saved itinerary so I can continue editing it.
- As a user, I want to edit an itinerary (notes/reorder/delete stops) so I can personalize my plan.
- As a user, I want to delete an itinerary so I can remove old plans.
- As a user, I want to log out so I can protect my account on shared devices.
- As a user, I want clear error messages for invalid email/password so I know how to fix it.
- As a user, I want my password to be stored securely so my account is protected.
